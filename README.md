## Overview

## How to Run

1. Fork and clone the repo
2. Must be using node v14 or higher
3. Run `npm install`
4. Run `npm start`
5. Go to http://localhost:3000 to view the application

## Description

Search and compare colleges: their fields of study, costs, admissions, results, and more.
